## UV 油物料完整清单

<!-- 来源: 哑光UV油标准书-20250730 (10)(1).xlsx §A+B区段 -->

### 1. 普通UV油

#### #UVC809（存仓物料）
- **名称**: 普通UV油
- **用途**: 咭书、硬盒、W-O书、骑钉书、挂历、银銻纸做纸袋、平装书封面
- **适用产品**: 同上
- **结构应用**: 单面V/双面V/封面V/书脊V/踩坑位X/内文V
- **适用界面**: 单粉纸/咭、粉灰咭、哑粉纸/咭、银銻纸、双粉纸/咭
- **印刷油墨面**: 普通油墨、LED-UV油墨、减喷粉快干油墨、耐晒油墨、快干油墨
- **后加工兼容**: 击凸V/击凹V/立体击凸V/深层击凸V/局部UV V/哑UV V/沟闪粉V/洒闪粉V/凸字油V/磨砂UV V/夜光油墨V/岩石UV V/普通植毛V/透明植毛V/闪粉植毛V/植毛压纹V/荧光毛V/机啤V/手啤V/正啤折V/反啤折V/啤切V/啤半穿V/压纹V/闸圆角V、大部分其他后加工为X
- **备注**: 普通用途由印刷部安排使用。书写白板笔转HUV-131UV油。

#### #PC-200（存仓物料）
- **用途/兼容性**: 与UVC809基本一致
- **备注**: 普通用途由印刷部安排使用。书写白板笔转HUV-131UV油。
- **差异点**: 植毛全部兼容（普通/透明/闪粉/压纹/荧光毛），UVC809长毛X而PC-200同

### 2. LED-UV油

#### #H-203（存仓物料）
- **名称**: LED-UV油
- **用途**: 咭书、硬盒、W-O书、骑钉书、挂历、银銻纸做纸袋、《金莎T8》工程、银銻纸过UV做咭书、平装书封面
- **适用界面**: 单粉纸/咭、粉灰咭、哑粉纸/咭、银銻纸、双粉纸/咭
- **印刷油墨面**: 普通油墨、LED-UV油墨、减喷粉快干油墨、耐晒油墨、快干油墨
- **后加工兼容**: 击凸V/击凹V/立体击凸V/深层击凸X/超级深层击凸X/击纹X/热辣X、啤类全部V、压纹X、局部UV X（不同于UVC809）、植毛X
- **限制**: 物料成分问题限制德国客户（Bonnier Germany）产品使用。其他客户产品由印刷部自行安排使用。
- **备注**: 订单管理科需于作业单注明非德国客户银銻纸咭书需使用H-203LED-UV油。

### 3. 免打底UV油

#### #7505（存仓物料）
- **名称**: 免打底UV油
- **用途**: 咭书、硬盒、W-O书、骑钉书、挂历
- **适用封面**: X（区别于普通UV油）
- **后加工兼容**: 啤类全部V、压纹X、烫金X、过胶X、丝印X、植毛X、印刷X、击凸/击凹全部X
- **备注**: 普通用途由印刷部安排使用。

### 4. 书袋专用半光哑UV油

#### #UVC830F6（跟JOB订购）
- **名称**: 书袋专用半光哑UV油
- **用途**: HM书袋产品
- **适用界面**: 单粉纸/咭、粉灰咭
- **后加工兼容**: 击凸V/击凹V/立体击凸V/深层击凸V、啤类全部V、其他烫金/过胶/丝印/植毛全部X
- **备注**: 订单管理科需于生产工程单上注明：因客户要求过UV后光泽度需在50-60度，故必须使用UVC830F6UV油。注意：此型号是用水油机的水油座来生产。

### 5. 高耐磨UV油

#### #UVC823（跟JOB订购）
- **名称**: 高耐磨UV油
- **用途**: 《金莎T8》工程、银銻纸过UV做咭书
- **适用界面**: 银銻纸（限定）
- **印刷油墨面**: LED-UV油墨（限定）
- **结构应用**: 内文X（仅此物料有此特性）
- **后加工兼容**: 击凸V/击凹V/立体击凸V、啤类全部V、其他全部X
- **备注**: 订单管理科需于生产工程单上注明：因高耐磨要求，德国客户银銻纸咭书必须使用UV油UVC823（跟单购买）。

### 6. 白板笔UV油

#### #AM-8828（跟JOB订购）
- **名称**: 白板笔UV油
- **用途**: PENUK01客户产品
- **适用界面**: 全部粉纸类、银銻纸
- **印刷油墨面**: 全部油墨类型
- **后加工兼容**: 啤类V、植毛全部V（普通/透明/闪粉/压纹/荧光毛/长毛）、压纹X、烫金X、过胶X、丝印X
- **备注**: 订单管理科需注明针对PENUK01客户：
  1) 双面过UV上折页机工程必须使用UV油AM-8828
  2) 可用于写白板笔（酒精性白板笔、水性白板笔）
  3) 针对UV面植毛工艺产品
  注意：非PENUK01客户且双面过UV上折书机工程（无须写白板纸）选用UVC819UV油。

#### #HUV-131（跟JOB订购）
- **名称**: 白板笔UV油
- **用途**: 写白板笔（酒精性白板笔、水性白板笔）；双面过UV后需写白板笔同时需要上折页机工程
- **兼容性**: 与AM-8828基本一致，植毛全部兼容
- **备注**: 
  1) 因需写白板笔必须使用UV油HUV-131
  2) 双面过UV后需写白板笔同时需要上折页机工程必须使用UV油HUV-131

### 7. 耐高温UV油

#### #UVC819（跟JOB订购）
- **名称**: 耐高温UV油
- **用途**: 需烫金边/银边/白边/机扫色边产品；双面过UV上折页机工程；UV面植毛工艺产品
- **后加工兼容**: 击凸V/击凹V/立体击凸V/深层击凸V、啤类全部V、压纹V（区别于其他UV油）、植毛全部V
- **备注**: 
  1) 需烫金边/银边/白边/机扫色边的产品必须使用UV油UVC819
  2) 非PENUK01客户且双面过UV上折书机工程（无须写白板纸）必须使用UV油UVC819
  3) 针对UV面植毛工艺产品

### 8. 低气味白板笔UV油

#### #PC-210（跟JOB订购）
- **名称**: 低气味白板笔UV油
- **用途**: 功能与AM-8828相同（仅气味降低）；针对气味敏感客户PENGUIN BOOKS使用
- **兼容性**: 与AM-8828基本一致
- **备注**: PC-210UV油能达到HUV-131UV油的所有功能。

### 9. 可烫金UV油

#### 东华UV EMBOSS光油（印刷中心自行开单）
- **名称**: 可烫金UV油
- **用途**: 需同时满足a)客人选用的金纸不适合做烫后过UV；b)客人不接受转用其他金纸；c)烫金线条细于3mm不适合作偷油处理；d)对烫金附着力要求不高
- **适用界面**: 单粉纸/咭、粉灰咭、哑粉纸/咭、银銻纸
- **印刷油墨面**: LED-UV油墨（限定）
- **后加工兼容**: 啤类全部V，其他全部X
- **备注**: 采购部收到Order需将"客人选用的金纸不适合烫后过UV"信息反馈给订单管理科。

### 10. 防爆色系列

#### #PC-8061 离线防爆色UV底油（存仓物料）
- **用途**: 平装书封面过UV油；扇形/风琴咭书类过UV工程；防爆色要求高的工程
- **适用界面**: 单粉纸/咭、粉灰咭、哑粉纸/咭、双粉纸/咭
- **后加工兼容**: 啤类V、其他全部X
- **备注**: 必须使用UV底油PC-8061配合现用UV(UVC809/PC-200UV油)离线过UV。

#### #PC-8062 离线抗爆线底油（跟JOB订购）
- **用途**: 手工毛布咭书、封面过UV且厚度小于6mm咭书、《开心乐园咭书系列》
- **备注**: 以上工程翻单需注明使用PC-8062底油+PC-216UV油进行生产。

#### #PC-216 抗爆UV油（跟JOB订购）
- **用途**: 同上，配合PC-8062使用
- **后加工兼容**: 击凸V/击凹V/立体击凸V/深层击凸V、啤类全部V、压纹V
- **备注**: 以上工程翻单需注明使用PC-8062底油+PC-216UV油进行生产。

## UV 油选用规则总结

### 按用途选型

| 用途 | 推荐型号 | 存仓/订购 |
|------|---------|-----------|
| 普通UV光油 | UVC809 / PC-200 | 存仓 |
| LED-UV油（普通） | H-203 | 存仓 |
| 免打底UV油 | 7505 | 存仓 |
| 书袋专用 | UVC830F6 | 跟JOB订购 |
| 高耐磨（德国客户） | UVC823 / H-203 | 跟JOB订购 |
| 白板笔书写 | HUV-131 / AM-8828 | 跟JOB订购 |
| 耐高温（烫金边/折页机） | UVC819 | 跟JOB订购 |
| 低气味（PENGUIN BOOKS） | PC-210 | 跟JOB订购 |
| 可烫金UV（特殊情形） | 东华UV EMBOSS光油 | 印刷中心自行开单 |
| 防爆色底油 | PC-8061 + UVC809/PC-200 | 存仓+跟JOB订购 |
| 抗爆线底油 | PC-8062 + PC-216 | 跟JOB订购 |

### 德国客户限制

H-203（LED-UV油）因物料成分问题，**限制德国客户（Bonnier Germany）产品使用**。德国客户银銻纸咭书必须使用UVC823（高耐磨UV油）。

### 白板笔书写选用规则

- 需写白板笔 → HUV-131
- PENUK01客户 → AM-8828
- PENGUIN BOOKS（低气味要求）→ PC-210（可达HUV-131同等功能）
- 非PENUK01客户，双面过UV上折书机（无须写白板笔）→ UVC819

### 植毛工艺选用

- UV面植毛工艺 → UVC819 / AM-8828 / HUV-131
- 普通UV油（UVC809/PC-200）植毛兼容：普通/透明/闪粉/压纹/荧光毛

### 后加工基本限制

UV油类物料普遍 **不兼容** 以下后加工：单双面印刷、油性油/水性油/UV油印刷、扫金粉（极少数除外）、逆向油、冷烫、仿皮革、凸字粉沟闪粉、烫金（普通/镭射/荧光/折光纹/立体/立体击凸）、过胶、丝印局部UV/哑UV（部分兼容）、皱纹UV/橘皮UV、粘配件/涂胶/辊金边等。

兼容的通常只有：啤类（机啤/手啤/啤折/啤切/啤半穿）、闸圆角、击凸/击凹/立体击凸（分物料）。
