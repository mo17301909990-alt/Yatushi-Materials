# 烫金后加工组合规则（后工序决策知识）

本文件编码"烫金+过胶"、"烫金+印刷UV"、"烫金+丝印LED UV洒闪粉"三种核心后加工组合的选用规则。

## 烫金+过胶组合规则

<!-- 来源: #N-BF-011-5 §过胶组合 -->

来源标准：《烫后加工："烫金 + 过胶"－组合应用指引书》。

决策步骤：从左到右看标准表格，先看适用界面 → 看过胶种类 → 看烫金过胶后加工 → 看烫金纸选用。相同条件都可选用时，任选一种即可。

### 过胶后含压力性加工：首选 G1 系列
烫金后过胶再加以下任意压力性加工，首选 G1 系列金纸：击凸、击凹、立体击凹凸、啤折、啤切、踩坑、压纹。在 CSI 系统选择对应型号即可，只看此标准就能完成选择。也可去《烫金：常用界面烫印性-指引书》确认界面/面积是否适用，但街货型号不同时可能需要确认。

### 过胶种类区分
水性光胶和水性哑胶是常见过胶材料。系统 lamination_compatibility 表记录过胶材质和过胶步骤的组合兼容规则。当前数据中过胶兼容表有 665 条记录。系统匹配时通过 laminationJoinCondition 关联：lc.product_type IS NULL OR lc.product_type = '' OR lc.product_type = p.hot_stamping_paper_type。

## 烫金+印刷 UV 组合规则

<!-- 来源: #N-BF-011-2 §印刷UV -->

来源标准：《烫后加工——"烫金 + 印刷UV"指引书》。

工艺流程为"先烫金后印刷 UV"时，先查看此标准。若此标准未能确定，再去《烫金：常用界面烫印性-指引书》选择对应的小型号。

方案选择：
- 需要紧身金纸 → S6 系列（当前数据库无 S6，但 SY6 有 36 个产品，S6 可能是旧命名）
- 需要印刷兼容 → E8 系列（有 1 个产品）

## 烫金+丝印LED UV洒闪粉组合规则

<!-- 来源: #N-BF-011-3 §丝印LEDUV -->

来源标准：《烫后加工："烫金 + 丝印LED UV洒闪粉"－组合应用指引书》。

此组合用于贺卡等需要洒闪粉效果的场景。决策时需先查此标准，确认金纸与丝印LED UV洒闪粉的兼容性。若烫金与丝印不重叠，也可以通过《烫金：常用界面烫印性-指引书》按界面选用。

## 先烫金后印刷的特殊判断

<!-- 来源: #N-BF-011-4 §印刷组合 -->

"先烫金后印刷"的判断方法：如果第一个工序是烫金，下一个工序是印刷，即判断为先烫金后印刷。此时打开标准《烫金+印刷－组合应用指引书》，再结合客户要求做选择。

## 布面纸烫金规则

<!-- 来源: #N-BF-010 §布面纸 -->

来源标准：《特殊界面烫金：布面纸烫印性－指引书》。

注意：此标准内的布面纸型号不齐全，大部分都没有。实际可用的型号为 JHF/JXF（与标准中 JHT 一样，只是 FSC 与非 FSC 的区别）。布面纸烫金基本上不分面积，标准也没有提到面积，且工程面积一般不大。

## 普通金纸/耐磨选择流程

<!-- 来源: #N-BF-013 §耐磨选用 -->

来源标准：《耐磨烫金纸选用指引书》。此标准有小部分普通耐磨/高耐磨，但实际型号不齐全。普通金纸不在此标准内。按步骤 1-7 筛选，走到第 7 步仍未确定时，转到《烫金：常用界面烫印性-指引书》。
