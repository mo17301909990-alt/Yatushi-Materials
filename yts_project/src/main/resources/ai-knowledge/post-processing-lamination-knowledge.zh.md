### 选用原则 — 水胶 vs 预涂

| 序号 | 一定要用水胶或首选水胶的产品类型 | 原因 |
|------|--------------------------------|------|
| 1 | 普通哑胶做立体烫金/(烫金+击凸重叠)且只能用普通烫金纸 | 减轻预涂膜与金纸摩擦导致甩金 |
| 2 | JetPress数码打印机做含皮壳结构过胶产品(精装书皮壳包纸、活页书支架)、板纸盒活动结构过胶产品 | 首选用水性光/哑胶，减少过胶离层发白风险 |
| 3 | 预涂膜(特殊膜)价格贵的工程 | 节省成本 |
| 4 | 其他特殊要求产品(高品质包装盒如CLB台历套装) | 预涂耐磨光胶静电大吸附尘点 |

### 版本更新要点

**关键变更记录**:
- V19: YT008A不能做丝印局部UV
- V20: 耐磨哑胶面对油面使用规则; YT008A/TN008不能烫金(部分金纸不能通过3M810测试)
- V21: 新增KPMB-F; YL30SR不能烫金
- V22: 删除YL30SR; 哑胶面烫金需结合烫金指引
- V23: 删除SSF01; HY65/HY40预涂光/哑胶停用
- V24: 新增水性膜要求(客服需在MO上注明)
- V25: 更新油墨适用性; 删除西班牙柔感哑胶(停产); 更新腾锦柔感哑胶
- **V26(当前)**: 删除预涂PET光胶(电晕值不稳定); 更新腾锦柔感哑胶; 更换耐磨光胶牌子/型号; 新增1208预涂光胶/1508预涂哑胶

### 过胶注意事项

- 预涂膜除水性单面电晕外均需要上胶水来覆膜（即当作即涂膜使用）
- 特殊功能膜的适用后加工参见厂内过胶膜指引
- 所有过胶膜不适用后加工：烫金(普通/镭射/荧光/折光等子类)—需根据指引筛选
- 普通光/哑胶不适宜直接植毛（除毛不干净）
- PET光胶不可用于双面过光胶粘接、天地盒机包盒以外的工程
- 双面过光胶且双面植毛的印张不适合做植毛

## 二、JP 750S印刷→后加工应用指引

<!-- 来源: JP 750S印刷后加工工艺标准书-20250808 (3).docx §全部 -->

**文件版次**: 04版（2025-08-08）
**目的**: 解决JP 750S打印稿及生产后，过胶粘接后出现离层问题，及印张表面张力低导致粘不上或丝印附着力问题。

### JP 750S水性油墨适用界面

**墨水**: JetPress 750S墨水
**待干时间**: ≧24小时（印刷标准环境下）
**适用纸张**: 单粉纸/双粉纸/单粉咭/双粉咭/哑粉咭/书纸(≧100g大货，≧80g打稿)/布面纸/漂白牛皮纸/原色牛皮纸/白面牛咭/胶底纸/花纸/胶片/牛油纸/铝箔纸

**限制**:
- 书纸<100g大货不适用（打印易起浪）
- 花纸：除粉纸/咭、书纸、牛皮纸、布面纸、布纹纸、牛油纸、胶底纸、板纸外的纸

| 过胶(水性膜) | ✓ 需≧36mN/m电晕值 |
| 过胶(预涂膜+胶水) | ✓ 需≧36mN/m，选合适预涂膜 |
| 过胶+击凸 | 需根据过胶指引选合适预涂膜 |

### JP过胶印张表面加工预防离层做法

**关键工艺**: 过胶前需要先过**传统丝印UV炉**（200~240 mJ/cm2），避免烫金、丝印加工用3M810胶纸测试后过胶离层。

**配合要求**:
- **水性膜**: 烫金/丝印局部UV/丝印UV沟闪粉 — 限制使用，过胶前必须过传统丝印UV炉
- **预涂膜**(上胶水覆膜): 烫金/丝印局部UV/丝印UV沟闪粉 — 限制使用，过胶前必须过传统丝印UV炉

**表面电晕值检测**: ≧36 mN/m 方可进行过胶

### 注意

- 其他特殊功能膜的适用后加工请参阅厂内过胶膜指引
- 预涂膜需上胶水覆膜（即当作即涂膜使用）
- 日后大货涉及的纸张、油墨、加工搭配不在上表时，需与NTD联系测试确认
