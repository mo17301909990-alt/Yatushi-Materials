# 烫金纸选用决策矩阵（参考标准 #N-BF-013）

本文件编码《烫金纸选用规则决策矩阵》（#N-BF-013）中的产品类型→适用界面→烫金类型→烫金纸选用的完整决策树。SmartVersion/Copilot 可根据此矩阵引导用户完成从产品类型到最终型号的推导。

## 决策入口：产品类型

<!-- 来源: #N-BF-013 §决策树 -->

用户需要先确定产品类型。系统 product_type_options 表记录以下类型：
- ID 1: 精平装/咭书
- ID 2: 贺咭/纸袋
- ID 3: 包装
- ID 5: 其他性能类型（高耐磨等）

## 精平装/咭书决策分支

<!-- 来源: #N-BF-013 §决策树 -->

精平装/咭书根据油墨体系和表面处理分为 4 个入口分支：

入口 1：普通油墨 + 6812/3301/440 水油 → 适用界面为印刷/过油面。此时查烫金类型→查图案面积→确定最终系列。

入口 2：GPO/HK90 LED UV 油 + GP/3301/004 + 油 → 适用界面为印刷/过油面。后续决策同上。

入口 3：预涂膜 → 适用界面为光膜面/过胶面。此时注意烫金在膜面上进行，需要考虑 ST 系列或其他过胶面适用系列。

入口 4：光膜 → 适用界面为光膜面。同上，需考虑膜面烫金的特性。

## 决策路径：从入口到系列

<!-- 来源: #N-BF-013 §决策树 -->

确定适用界面后，按以下顺序推导：
1. 查耐磨烫金纸选用指引书（步骤 1-7），看能否确定纸性能类型（普通金纸/普通耐磨/高耐磨）
2. 若步骤 7 仍未确定 → 转到《烫金：常用界面烫印性-指引书》，按界面+表面+图案确认系列
3. 涉及后加工 → 转到对应后加工组合指引书：
   - 烫金+过胶 → 过胶组合指引书
   - 烫金+印刷UV → 印刷UV组合指引书
   - 烫金+丝印LED UV洒闪粉 → 丝印LED UV洒闪粉指引书

## 产品类型与高性能匹配

<!-- 来源: #N-BF-013 §耐磨选用 -->

product_type_id = 5 时启用高性能（高耐磨/普通耐磨）优先排序配置。该配置由 product_type_sort_config 表控制。当用户选择高耐磨或普通耐磨时，系统会优先展示对应性能类型的产品，与系列无关。数据库产品按 hot_stamping_paper_type 字段区分为：普通金紙（554 个）、鐳射燙金紙（137 个）、普通耐磨（102 个）、高耐磨（4 个）。

## 适用界面(前工序)选项

<!-- 来源: #N-BF-009 §常用界面 -->

系统 pre_processing_steps_options 表记录了所有适用界面/前工序选项。用户匹配时需先选择适用的前工序，相当于#N-BF-013 中的"适用界面"维度。SmartVersion/Copilot 应引导用户先确认前工序/适用界面，再进入后续决策。
