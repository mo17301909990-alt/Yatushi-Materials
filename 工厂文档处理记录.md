## 处理规则

### 1. 文件解析

- **.docx**: 用 python-docx 提取段落+表格
- **.xlsx**: 用 openpyxl 读取 sheet/行/列

### 2. 知识库生成

输出到 `yts_project/src/main/resources/ai-knowledge/`，格式：

```markdown
# 标题

## 章节

<!-- 来源: 原始文件名 §章节位置 -->

正文内容...

| 表格列1 | 表格列2 |
|---------|---------|
| 数据    | 数据    |
```

### 3. 数据入库（可选）

如果文件包含结构化兼容性数据，除知识库外还应：
- 更新对应 CRUD 模块的数据（Service/Controller/Mapper/Model）
- 准备 SQL 脚本或导入工具

### 4. 验证

- 编译后端 `mvn compile` 无错误
- 构建前端 `npm run build` 无 TypeScript 错误
