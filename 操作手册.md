# 后台管理操作手册

---

## 用户管理

**路径**: `/admin/user-management`

### 新增用户
1. 点击"新增用户"按钮
2. 填写以下信息：
   - **用户名**（必填，唯一）
   - **邮箱**（必填）
   - **初始密码**（必填）
   - **角色**（必填，可多选：ADMIN / USER / OPERATOR）
3. 点击"保存"

### 编辑用户
1. 点击用户列表中的"编辑"按钮
2. 修改：用户名、邮箱、状态（启用/禁用）
3. 如需重置密码，点击"重置密码"按钮
4. 点击"保存"

### 分配角色
1. 点击用户列表中的"角色分配"按钮
2. 在角色列表中勾选要分配的角色
3. 点击"保存"

### 删除用户
1. 点击用户列表中的"删除"按钮
2. 确认删除操作

---

## 角色管理

**路径**: `/admin/role-management`

### 新增角色
1. 点击"新增角色"按钮
2. 填写以下信息：
   - **角色名称**（必填，如：测试员）
   - **角色键**（必填，英文唯一，如：TESTER）
   - **描述**（选填）
   - **状态**（启用/禁用）
3. 点击"保存"

### 配置角色权限
1. 点击角色列表中的"权限配置"按钮
2. 在权限树中勾选需要的权限（可按模块筛选）
3. 点击"保存"

**常用权限说明**：

| 权限键 | 说明 |
|--------|------|
| `user:view` | 查看用户 |
| `user:create` | 新增用户 |
| `user:edit` | 编辑用户 |
| `user:delete` | 删除用户 |
| `role:view` | 查看角色 |
| `role:create` | 新增角色 |
| `role:edit` | 编辑角色 |
| `role:delete` | 删除角色 |
| `process:hotstamping:view` | 查看烫金工艺 |
| `process:hotstamping:edit` | 编辑烫金工艺 |
| `material:rule:view` | 查看物料规则 |
| `material:rule:edit` | 编辑物料规则 |
| `matching:tag:select` | 标签选择 |
| `matching:result:view` | 查看匹配结果 |
| `system:log:view` | 查看操作日志 |

### 分配用户到角色
1. 点击角色列表中的"分配用户"按钮
2. 在用户列表中勾选要分配的用户
3. 点击"保存"

### 编辑角色
1. 点击角色列表中的"编辑"按钮
2. 修改：角色名称、描述、状态
3. 点击"保存"

### 删除角色
1. 点击角色列表中的"删除"按钮
2. 确认删除操作

## 权限管理

**路径**: `/admin/permission-management`

## 烫金物料信息管理

**路径**: `/admin/material/hot-stamping-material`

### 说明
管理烫金纸（Hot Stamping Paper）产品信息、规格信息和价格信息。

### 搜索筛选
可按以下条件搜索：
- **关键词**：搜索烫金纸系列、型号、物料编号、Leo Touch编号、SPNO、颜色
- **烫金纸性能类型**：筛选烫金纸性能类型
- **Leo Touch编号**：搜索Leo Touch编号
- **SPNO**：搜索SPNO
- **牌子**：搜索牌子
- **颜色**：搜索颜色

### 新增产品
1. 点击"添加产品"按钮
2. 填写表单：
   - **烫金纸系列**（必填）：如 K508FX
   - **型号**（必填）：如 K508FX-000
   - **物料编号**（必填）：唯一编码
   - **烫金纸性能类型**：如 K系列
   - **牌子**：如 KOLON
   - **封度**：如 12μm
   - **单位**：如 卷
   - **是否常用**：常用/不常用/跟JOB
   - **是否街货**：勾选表示是街货
   - **物料状态**：可用/废弃
   - **说明**：产品描述
   - **概述**：产品概述
3. Leo Touch信息：Leo Touch编号、SPNO
4. 规格信息：颜色、尺寸、松紧度、性能、温度范围
5. 价格信息：价格
6. 点击"添加"

### 编辑产品
1. 在产品列表中点击"编辑"按钮
2. 修改需要更改的字段
3. 点击"保存"

**注意**：根据权限不同，某些字段可能为只读状态（灰色显示）

### 查看详情
1. 在产品列表中点击"详情"按钮
2. 弹出详情对话框，显示完整的产品信息

### 删除产品
1. 在产品列表中点击"删除"按钮
2. 确认删除操作

### 批量修改
1. 勾选列表左侧的复选框，选择要修改的产品（可多选）
2. 点击"批量修改"按钮
3. 填写要修改的字段（留空的字段不会修改）
4. 点击"确认修改"

### 批量删除
1. 勾选要删除的产品
2. 点击"批量删除"按钮
3. 确认删除操作

### 导出数据
1. 点击"导出数据"按钮
2. 系统导出Excel文件并自动下载

### 导入数据
1. 点击"导入数据"按钮
2. 选择Excel文件（.xlsx 或 .xls）
3. 选择冲突处理策略：跳过冲突数据/覆盖现有数据/创建新记录
4. 如选择"覆盖现有数据"，可勾选"导入前备份"
5. 点击"开始导入"
6. 查看导入结果（总记录数、新增数、更新数、失败数）

### CSV批量更新
1. 点击"CSV 批量更新"按钮
2. 点击"下载 CSV 模版"获取模板
3. 准备CSV文件：必须包含 `id` 列，其余列名为英文camelCase
4. 选择CSV文件，点击"上传并更新"
5. 查看更新结果

### 权限说明

| 权限 | 可操作范围 |
|------|-----------|
| 管理员 | 可编辑全部字段 |
| 燙金物料-技術（全部欄位） | 可编辑全部字段 |
| 燙金物料-採購（價格+狀態） | 只能编辑价格和物料状态 |
| 燙金物料主數據 | 可编辑全部字段 |

### 字段说明

| 字段 | 说明 |
|------|------|
| 烫金纸系列 | 产品名称/系列名，如 K508FX |
| 型号 | 产品型号 |
| 物料编号 | 仓库物料编码，唯一 |
| Leo Touch编号 | 公司内部编号 |
| SPNO | 产品销售编号 |
| 封度 | 纸张厚度，如 12μm |
| 是否街货 | 是否为常备库存货品 |
| 物料状态 | 可用/废弃 |

## 通用界面（前工序）管理

**路径**: `/admin/material/pre-processing-interface`

### 说明
管理前工序适用界面的配置和选项。

### 搜索筛选
- **前工序类型**：下拉选择
- **状态**：激活/停用
- **工艺类型**：下拉选择
- 点击"**搜索**"按钮查询

### 新增前工序
1. 点击"**添加前工序**"按钮
2. 填写表单：
   - **前工序名称**（必填）：如 烫金
   - **工艺**：如 热压
   - **状态**：激活/停用
   - **前工序类型**：如 测试、过胶面、印刷/过油面
   - **步骤说明**：描述该工艺的具体步骤
   - **注意事项**：选择关联的注意事项
3. 点击"**添加**"

### 编辑前工序
1. 在前工序列表中点击"**编辑**"按钮
2. 修改需要更改的字段
3. 点击"**保存**"

### 删除前工序
1. 在前工序列表中点击"**删除**"按钮
2. 确认删除操作

### 配置映射
1. 在前工序列表中点击"**配置映射**"按钮
2. 进入映射配置页面
3. 设置与产品/物料的关联关系
4. 保存配置

### 切换状态
- 点击"**激活**"将停用的前工序启用
- 点击"**停用**"将激活的前工序禁用

### 批量切换状态
1. 勾选左侧复选框，选择要操作的前工序
2. 点击"**批量切换状态**"按钮
3. 确认操作

### 调整显示顺序
- 在列表中直接修改**显示顺序**的数字
- 或使用 **上下箭头** 调整顺序
- 修改后自动保存

### 导出数据
- 点击"**导出前工序**"：导出前工序列表
- 点击"**导出全部**"：导出全部配置映射

### 查看所有配置映射
1. 点击"**查看所有配置映射**"按钮
2. 在弹窗中查看所有前工序的配置映射关系

## 烫金图案基础信息管理

**路径**: `/admin/material/hot-stamping-pattern`

### 说明
管理烫金图案的基础信息，包括图案类型、线条粗细范围、实地面积范围等配置，用于烫金工艺的兼容性判断和参数设置。

### 搜索筛选
- **选项名称**：输入图案名称搜索
- **图案类型**：输入图案类型搜索
- **状态**：全部状态/激活/未激活
- **排序方式**：排序顺序/选项名称/图案类型/创建时间
- 点击"**筛选**"按钮查询

### 新增图案
1. 点击"**添加图案**"按钮
2. 填写表单：
   - **选项名称**（必填）：如 烫金图案A
   - **图案类型**：如 实地、线条、混合
   - **线条粗细范围**：最小值(mm)和最大值(mm)
   - **实地面积范围**：最小值(mm²)和最大值(mm²)
   - **排序顺序**：数字越小排序越靠前
   - **激活状态**：勾选表示激活
   - **注意事项**：选择关联的注意事项
3. 点击"**添加**"

### 编辑图案
1. 在图案列表中点击"**编辑**"按钮
2. 修改需要更改的字段
3. 点击"**保存**"

### 复制图案
1. 在图案列表中点击"**复制**"按钮
2. 系统创建一个相同的副本，进入编辑模式
3. 修改名称等信息后保存

### 配置映射
1. 在图案列表中点击"**配置映射**"按钮
2. 设置与烫金类型或其他配置的映射关系
3. 保存配置

### 删除图案
1. 在图案列表中点击"**删除**"按钮
2. 确认删除操作

### 批量操作
1. 勾选左侧复选框，选择要操作的图案（可多选）
2. 可执行：
   - **批量编辑**：修改选中项的共同字段
   - **批量删除**：删除所有选中的图案
   - **批量设置注意事项**：为选中的图案设置注意事项
3. 点击"**清除选择**"可取消选择

### 导出数据
1. 点击"**导出数据**"按钮
2. 系统导出Excel文件并自动下载

### 视图切换
- **表格视图**：以表格形式显示数据
- **卡片视图**：以卡片形式显示数据
- 点击页面右上角的视图切换按钮切换

### 字段说明

| 字段 | 说明 |
|------|------|
| 选项名称 | 烫金图案的名称 |
| 图案类型 | 图案的分类，如实地、线条、混合 |
| 线条粗细范围 | 图案线条的粗细范围，单位mm |
| 实地面积范围 | 图案实地的面积范围，单位mm² |
| 排序顺序 | 显示顺序，数字越小越靠前 |

## 适用界面管理

**路径**: `/admin/material/applicable-interface`

### 说明
管理布面纸类型和兼容性配置。支持配置系列优先级规则（档位与系列名可任意配置，不限于 SK、L817）。

---

### 系列优先级规则

#### 新建规则
1. 点击"**新建规则**"按钮
2. 填写：
   - **规则名称**（必填）：如 SK系列优先级
   - **排序顺序**：数字越小排序越靠前
   - **状态**：启用/停用
3. 点击"**保存**"

#### 配置档位
1. 在规则列表中点击"**配置档位**"按钮
2. 进入档位配置页面
3. 可进行：
   - **新增档位**：设置优先级数值和系列名称
   - **编辑档位**：修改已存在的档位
   - **删除档位**：移除不需要的档位
   - **调整顺序**：拖动或使用箭头调整档位顺序
4. 配置完成后保存

#### 编辑规则名称
1. 在规则列表中点击"**编辑名称**"按钮
2. 修改规则名称
3. 保存

#### 删除规则
1. 在规则列表中点击"**删除**"按钮
2. 确认删除操作

---

## 矩阵表导出导入

### 路径
`物料规则管理 → 矩阵表导出导入`

### 功能说明
按指引書對應的矩陣表進行 Excel / Word 導出與導入，用于批量管理各類配置數據。

### 界面说明

页面列出7種矩陣表類型：

| 序号 | 矩阵表名称 | 说明 | 对应文档 |
|------|-----------|------|---------|
| 1 | 耐磨燙金紙選用指引書 | 產品類型、圖案特徵、燙金類型、前工序及耐磨燙金紙映射配置 | 《耐磨燙金紙選用－指引書》 |
| 2 | 常用界面燙印性 組合應用表 | 燙金紙－常用界面燙印性（類型、系列、燙金圖案、燙金類型、常用界面、特殊界面） | 《燙金：常用界面燙印性-指引書》 |
| 3 | 「布面紙+燙金」－組合應用表 | 特殊界面布面紙與燙金紙系列的燙印性 | 《特殊界面燙金：布面紙燙印性－指引書》 |
| 4 | 燙金後過膠－應用限制 | 後加工序燙金後過膠矩陣 | 《燙後加工："燙金 + 過膠"－組合應用指引書》 |
| 5 | 燙金 + 印刷UV | 印刷UV 配置 | 《燙後加工－「燙金 + 印刷UV」指引書》 |
| 6 | 燙金 + 印刷 | 印刷配置 | 《燙後加工－「燙金 + 印刷」指引書》 |
| 7 | 燙金 + 絲印LED UV灑閃粉 | 配對燙金紙型號矩陣 | 配對燙金紙型號（燙金 + 絲印LED UV灑閃粉） |

**每種類型支持的操作按鈕：**
- **導出 Excel**（綠色按鈕）：下載 Excel 格式的矩陣表
- **導出 Word**（藍色按鈕）：下載 Word 格式的矩陣表
- **導入**（黃色按鈕）：從 Excel/Word 文件導入數據

### 导出操作

1. 點擊想要導出的矩陣表對應的**導出 Excel** 或 **導出 Word** 按鈕
2. 瀏覽器自動下載文件，文件名包含矩陣表名稱和日期

### 导入操作

1. 點擊想要導入的矩陣表對應的**導入**按鈕
2. 彈出導入對話框，可選：
   - **下載 Word 模板（帶舊數據）**：先下載包含現有數據的 Word 模板，修改後再上傳
3. 選擇要導入的 **.xlsx 或 .docx 文件**
4. 選擇**衝突處理**策略：
   - **跳過已有記錄**：不覆蓋已存在的數據
   - **覆蓋已有記錄**：用新數據替換舊數據
5. 點擊**開始導入**
6. 查看導入結果：
   - **新建**：新建的記錄數
   - **更新**：更新的記錄數
   - **跳過**：跳過的記錄數
   - **錯誤**：如有錯誤，顯示錯誤詳情

## 注意事項字典管理

### 路径
`物料规则管理 → 注意事項字典管理`

### 功能说明
管理所有规则匹配时需要显示的注意事项，可包含富文本内容（表格、图片等），供其他模块关联使用。

### 界面说明

**搜索/筛选区域：**
- **搜索框**：输入编码、标题或描述进行模糊搜索
- **分类筛选**：下拉选择分类筛选

**数据表格列：**
| 列名 | 说明 |
|------|------|
| ID | 记录ID |
| 編碼 | 注意事项唯一编码（如：PRINT_001） |
| 標題 | 注意事项标题 |
| 問題描述 | 简要问题描述 |
| 解決方案 | 简要解决方案 |
| 分類 | 所属分类（印刷/過膠/燙金/前工序/絲印） |
| 優先級 | 优先级（数字越大优先级越高） |
| 狀態 | 启用/停用状态 |
| 富文本 | 是否有富文本内容（有则显示"預覽"按钮） |
| 操作 | 编辑、删除 |

### 新增/编辑注意事项

1. 点击**新增注意事項**按钮
2. 填写表单：
   - **編碼***：必填，输入唯一编码
   - **標題**：可选，标题
   - **分類**：下拉选择或输入新分类
   - **優先級**：数字0-10，越大越高
   - **狀態**：启用/停用开关
   - **問題描述**：可选，简要描述问题
   - **解決方案**：可选，简要解决方案
   - **富文本內容**：支持插入表格、图片等（可选）
3. 点击**保存**

### 富文本编辑器

点击**富文本內容**区域可以：
- 编辑带格式的详细内容
- 插入图片（本地图片自动转为Base64）
- 插入表格
- 支持复制粘贴网页内容

### 预览

点击富文本列的**預覽**按钮可查看完整的富文本内容。

### 分类说明

| 分类 | 标签颜色 |
|------|---------|
| 印刷 | 黄色 |
| 過膠 | 蓝色 |
| 燙金 | 绿色 |
| 前工序 | 紫色 |
| 絲印 | 红色 |

## 操作日志

### 功能说明
查看谁在何时对哪块数据做了何种操作，便于审计与追溯。

### 界面说明

**筛选条件：**
- **模块**：下拉选择，可选：
  - 燙金物料信息
  - 導入備份與恢復
  - 產品類型管理
  - 燙金圖案(耐磨燙金紙選用)
  - 燙金類型管理
  - 適用界面(前工序)
  - 適用界面管理
  - 燙金圖案(常用界面選用)
  - 燙后加工(印刷)
  - 燙后加工(UV)
  - 燙后加工(LED UV)
  - 耐磨金纸映射
  - 注意事項字典
- **操作类型**：下拉选择（导入/恢复/新增/修改/删除）
- **时间起**：选择开始时间
- **时间止**：选择结束时间
- **查询/重置**：执行筛选或清空条件

**数据表格列：**
| 列名 | 说明 |
|------|------|
| 操作人 | 进行操作的用户 |
| 操作时间 | 操作发生的具体时间 |
| 操作类型 | 导入/恢复/新增/修改/删除 |
| 模块 | 所属功能模块 |
| 对象类型 | 操作对象的类型 |
| 对象描述 | 操作对象的描述 |
| 变更摘要 | 变更的简要说明 |
| 变更详情 | 变更的详细信息 |
| IP | 操作者的IP地址 |

**分页：**
- 显示在表格底部
- 每页显示20条记录
- 支持翻页查看更多记录

**空状态提示：**
"暂无操作记录。在管理端进行导入、恢复等操作后会自动记录。"
