# /grill — 代码审查

**用法**: `/grill <scope> [--fix]`

| 参数 | 说明 |
|------|------|
| `scope` | `all` 全量 / `diff` 当前改动 / `path` 指定文件 |
| `--fix` | 审查后自动修复发现的问题 |

## 审查维度

1. **正确性** — 逻辑缺陷、空指针、事务问题
2. **安全** — SQL 注入、权限绕过、硬编码密钥
3. **性能** — N+1 查询、缺失索引、大对象传输
4. **质量** — 重复代码、异常处理、日志覆盖

## 执行

```bash
# 审查当前 diff
/grill diff

# 审查全部代码
/grill all

# 审查并修复
/grill diff --fix
```
