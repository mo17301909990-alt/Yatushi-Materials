# 数据质量审计报告

**生成时间**: 2026-06-13
**审计范围**: 16个P0模块，106条产品记录，6470条兼容记录
**审计方式**: 数据库深度扫描 + 交叉核验 + 计数对比

---

## 1. Executive Summary

本次审计覆盖了全部16个印刷厂资料P0模块，涉及106种物料产品和6470条后加工兼容性记录。总体而言，数据入库完整性良好（0缺失表、0孤立兼容记录），但数据质量存在三类系统性问题：**繁体/简体混用**导致跨模块查询断裂（烫金类型1345条记录存在繁简变体）；**过胶材料模块字段污染**严重（responsible_person列被误存为厚度值，category/color/usage_text列100%为空，且实测场景6个API请求全部返回空结果）；**部分小模块存在100% NULL字段**（发光油墨的supplier_code/min_sheet_size/min_spacing等列全部为空）。此外，audit预期计数与DB实际计数在7个模块中存在差异（共计相差8条记录），主要源于parser对重复行/变体行的计数策略不同。综合评估数据健康评分为 **C级（需整改）**。

## 2. 重复数据分析

### 2.1 跨模块重复

| 物料编码 | 物料名称 | 出现在 | 出现次数 | 裁决 | 原因 |
|---------|---------|--------|---------|------|------|
| #CS-220902 | 絲印UV光油 | silicone_glittering_star_product(id=2), silicone_screen_uv_product(id=6) | 2 | **保留双方** | 分属不同工艺模块（闪星 vs 丝印UV），各有独立FK作用域的兼容表，且携带不同的模块特有元数据（闪星归类为"油類/透明色"，丝印UV有详细的尺寸规格和库存编号） |

**结论**: 无需要合并或删除的重复记录。该条"重复"实为合理的模块间共享物料，保留了各自模块的上下文信息。

### 2.2 模块内重复

各模块内部无完全重复的行（相同material_code + material_name的组合没有重复出现）。部分模块（如uv_oil_matte）存在material_name相同但usage不同的变体行，这是有意为之的业务变体，不属于数据重复。

**分析**: 大多数模块V率在39%-47%之间，分布均匀。珠光云母和压光UV油仅各1条记录无参考价值。过胶材料的V率(46.9%)在正常范围内，说明兼容数据本身可能正确，问题出在**产品元数据（usage/category/color）的缺失**，导致前端筛选条件无法匹配任何结果。

### 4.2 步骤名称一致性问题

**严重问题 SN-01: 繁简混用（级别：HIGH）**

hot_stamping_type列存在系统性繁简混用，4对语义完全相同的值因字符编码不同被存储为不同值：

| 简体变体 | 简体次数 | 繁体变体 | 繁体次数 |
|---------|---------|---------|---------|
| 平面烫金-到邊位 | 863 | 平面燙金-到邊位 | 452 |
| 平面烫金-於中間位 | 225 | 平面燙金-於中間位 | 30 |
| 折光烫金 | 69 | 折光燙金 | 3 |
| 磨砂烫金 | 78 | 磨砂燙金 | 3 |

**影响**: 总计1235条简体 vs 715条繁体。当查询条件传递简体值时，繁体记录无法命中，反之亦然。这会导致前端兼容性查询结果缺失约36.7%的记录。

**问题 SN-02: 跨模块步骤集不一致（级别：MEDIUM）**

- UV油哑油和水油哑油的71个步骤完全对齐（完美匹配）
- 硅胶模块有81个步骤（比UV多10个）：UV色墨、裱紙、打孔、袋底/（正折）、袋底反折位、袋筒/風琴位/（反折）、袋筒/風琴位/（正折）、逆向UV、普通油、折袋口/（正折）

**原因**: 硅胶模块覆盖了制袋/制盒的额外工艺步骤，这些步骤在UV油/水油场景中不存在。

### 4.3 物料编码格式一致性

- 16个模块中，物料编码格式存在差异：`#CS-` 前缀、"`UV-`" 前缀、纯数字等不同编码体系的混用
- 同一模块内编码格式基本一致，但跨模块检索时无法通过编码格式统一识别

## 7. 模块健康评分

评分标准:
- **A**: 数据完整，无NULL关键字段，计数一致，API测试通过
- **B**: 数据基本完整，少量非关键NULL，计数差异≤1
- **C**: 存在关键字段NULL或字段污染，计数差异≥2
- **D**: 大量数据缺失或错误，功能不可用

### 整体评分: **C**

**加权平均**: (C*2 + C*1 + B*11 + D*1 + B*1) / 16 = 4个非A模块，其中2个C、1个D

**总结**: 11个B级模块满足基本使用条件（数据完整度>80%），但2个C级模块（发光油墨、白UV）和1个D级模块（过胶材料）需要优先修复。API实测6/6空结果说明端到端链路需要紧急排查，该问题优先级高于所有数据修复。

### A. 审计命令

```sql
-- 跨模块重复
SELECT material_code, material_name, COUNT(*) 
FROM (
  SELECT material_code, material_name, 'silicone_glow_ink' as src FROM silicone_glow_ink_product
  UNION ALL
  SELECT material_code, material_name, 'silicone_white_uv' FROM silicone_white_uv_product
  -- ... (全16表 UNION)
) all_products
GROUP BY material_code, material_name
HAVING COUNT(*) > 1;

-- NULL分析
SELECT 'table_name' as tbl, column_name, 
  COUNT(*) as total,
  SUM(CASE WHEN column_name IS NULL THEN 1 ELSE 0 END) as null_count
FROM information_schema.columns;

-- 兼容性分布
SELECT compatibility_status, COUNT(*) 
FROM {module}_compatibility 
GROUP BY compatibility_status;
```
